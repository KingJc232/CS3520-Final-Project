# Tic Tac Toe

### Goal: 
To Create Tic Tac Toe Game and a Symbolic AI using the Min Max Algorithm with Alpha Beta Pruning

### Software Used: 
  - Python 3.8
  - Pygame
  
### Algorithms Used:
  - MiniMax Algorithm with Alpha Beta Prunning
